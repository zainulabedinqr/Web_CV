T.NODE: Introduction

T. Node is an innovative tool that automates public transit network design, offering a wide range of feasible configurations for transit authorities and consultancies. It reduces manual effort, accelerates planning, and provides both quantitative and qualitative evaluations to support strategic decision-making.

T.NODE: Network Loading

This T. Node interface allows users to import datasets such as road networks, demand matrices, and stop locations into the model.

T.NODE: Parameter Setting

This interface is divided into three sub-sections: route creation and fleet mix parameters, public transport network generation and optimization, and demand assignment functionality.

T.NODE: Results

This interface consists of two components: an evaluation dashboard for visualizing & comparing optimized public transport networks based on performance metrics, and an export module for downloading networks as .txt or PTV Visum-compatible formats for further planning.
