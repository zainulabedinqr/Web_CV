MOVE: Monitoring

It enables real-time tracking of active bus assets within the network, identifying each bus by its route, direction, and block id. It provides live ETA estimates for all stops, additionally, all live tracking data is stored for post-event analysis and playback, enhancing operational insights and decision-making.

MOVE: On-Time Performance

It evaluates the on-time performance of the entire network and individual routes, identifying the best and worst-performing routes and stops. It offers multiple views, including a timeline, day-of-week and hourly analysis,providing comprehensive insights into service reliability.

MOVE: Public Transit Network Analysis

It rovides a high-level overview of public transit performance, showcasing key metrics such as ridership, fare collection, reliability, and speed across the network. It identifies the best and worst-performing routes, offering a bird’s-eye view for assessing overall system performance.

MOVE: Public Transit Route Analysis

It delivers a detailed analysis of route-level performance, encompassing metrics from the network level alongside route topology, ridership, average speed, temporal boarding levels, on-time performance, farebox recovery and service losses, providing comprehensive route performance insights.

MOVE: Public Transit Stop Analysis

MOVE offers stop-level insights, including key metrics, ridership data, and the route-wise ridership share for each stop. It features an interactive map view and provides detailed information about each stop, such as its type, amenities, and associated attributes, enabling comprehensive analysis and visualization.

MOVE: AI Mobility Mind
